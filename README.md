# Sister Sister Baked Goods Website

## Overview
This is the official website for **Sister Sister**, a baking company specializing in custom baked goods. The website allows customers to learn more about the business, submit inquiries, and place custom orders online.

## Features
- Home page introducing Sister Sister Bakery  
- About page with business story and information  
- Order form for custom cake and baked good requests  
- Inquiry form for general questions and event requests  

## Pages
- **Home** – Landing page with highlights of the bakery  
- **About** – Background and story of Sister Sister  
- **Order** – Form for placing custom orders  
- **Contact/Inquiry** – Form for questions and special requests  

## Technologies Used
- HTML  
- CSS  
- JavaScript (if applicable)

## Forms
### Order Form
Customers can submit custom order requests including:
- Name  
- Contact information  
- Order details  
- Date needed  
- Special instructions  

### Inquiry Form
Used for general questions, catering requests, or collaborations.

## Purpose
The website provides an easy way for customers to connect with Sister Sister Bakery, place orders, and make inquiries.

## Setup
1. Clone the repository  
2. Open `index.html` in a browser  
3. Or deploy using GitHub Pages / hosting platform of choice
